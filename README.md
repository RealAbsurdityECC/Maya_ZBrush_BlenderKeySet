# Maya_ZBrush_BlenderKeySet
A custom keyset for Blender that mimics Maya in Object and Edit modes and Zbrush in Sculpt mode.

The Maya keybindings are approximately **85% similar** to their Maya inspiration, covering the most commonly used shortcuts for object manipulation, editing, and navigation. The ZBrush keybindings are approximately **90% similar** to their ZBrush inspiration, closely replicating the sculpting workflow and brush controls.
